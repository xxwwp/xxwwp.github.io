## 零碎的

### 编译器和解释器

- 编译器

  获取源代码输入，生成可执行文件，机器码或者汇编等。比如 C/C++、rust 的编译器

- 解释器

  获取源代码输入，返回输出，比如 PHP。

### 核心步骤

1. Lexical Analysis 词法分析
1. Parsing 解析
1. Semantic Analysis 语义分析
1. Optimization 优化
1. Code Generation 代码生成
